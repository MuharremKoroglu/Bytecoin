## <h2> About This App <img src = "https://c.tenor.com/JsoERRQcZqYAAAAi/thumbs-up-joypixels.gif" width = "42"></h2>
* With this application made using SwiftUI, users can follow cryptocurrency exchanges, buy and sell crypto, and also create their own watch lists and portfolios.
* Used Alamofire library to handle all network requests.
* While the cryptocurrency data used in the application is provided by the CoinGecko API, fake user information is taken from the Random User Generator API.
* Firebase products Anonymous Authentication and Cloud Firestore are used in the application. In this way, user data is stored more securely.
* Animations were used on many screens with LottieFiles to increase the in-app user experience.
* Paging technique was used for the received cryptocurrency market data, aiming for fewer network requests and faster data loading.
* Generic structures have been used as much as possible in the application, and code crowding has been tried to be reduced by injecting data between screens.
